# VaultForge

A Bash based steganography toolkit.

Hide files inside images and videos with optional AES-256 encryption.

## Tools

### Image Vault

Hide files inside images.

Run:

bash ivault.sh


### Video Vault

Hide files inside videos.

Run:

bash vvault.sh


## Features

- Hide files inside images
- Hide files inside videos
- Extract hidden files
- AES-256 encryption
- Termux support


## Requirements

- Bash
- OpenSSL
- File utility
